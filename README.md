## Run Locally

Clone the project

```bash
  git clone https://github.com/micheldore/songsrated.git
```

Go to the project directory

```bash
  cd songsrated
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
